# React SSR Basics Example

An example of a server-side rendered (SSR) React application. This project demonstrates fundamental SSR features,
including:

- Implementation SSR
- Routing
- Production-Ready Setup

## Run Project

Install Dependencies:

```shell
pnpm install
```

Development Mode:

```shell
pnpm dev
```

Production Mode:

```shell
pnpm build && pnpm start
```
